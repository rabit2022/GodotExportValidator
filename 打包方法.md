
## 打包nuget包

1. 编译 SourceGenerator

- 排除[godot-export-validator](godot-export-validator),[GodotTester](GodotTester)
- 编译,Tester 报错正常,忽略

2. 编译 godot-export-validator项目

godot 打开项目,编译成功即可

3. 打包nuget包

# 跳过编译阶段
```bash
dotnet pack Generator.csproj -c Release  --no-build

```

## 查看测试

### Tester
[Tester](Tester) 是 [Generator](Generator) 的测试项目,测试代码生成是否正确
[generated](Tester/obj/Release/net8.0/generated) 可以看到生成的代码

项目配置:
```xml
    <EmitCompilerGeneratedFiles>true</EmitCompilerGeneratedFiles>
```
### GodotTester
[GodotTester](GodotTester) 是 [godot-export-validator](godot-export-validator) 的测试项目

用godot打开项目,编译运行即可
